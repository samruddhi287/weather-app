# 🌤 Weather App

A modern Weather Application built using HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API.

## 🚀 Features

- Search weather by city name
- Real-time temperature display
- Weather condition display
- Humidity information
- Responsive design
- User-friendly interface
- API integration using Fetch API

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
